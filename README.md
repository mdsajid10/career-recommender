# 🎯 AI-Powered Career Recommender

An AI-based career recommendation system built with **FastAPI**.  
It suggests careers based on your skills and interests using simple matching logic (Week 1).  
Later weeks will enhance it with **ML models, Hugging Face embeddings, OpenAI, and a React dashboard** 🚀  

---

## ⚡ Features (Current Progress)
- ✅ FastAPI backend with `/recommend` endpoint  
- ✅ Dataset of 20+ careers with skills + interests  
- ✅ Weighted scoring system (skills > interests)  
- ✅ Swagger UI testing at `/docs`  
- 🔜 Smarter recommendations with ML & embeddings
